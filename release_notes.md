# Sandbox Meteor v1.0 – Ecosystem Closed Loop

## 🎯 版本亮点
- **全链条闭合**：雷暴-生态-政策模拟闭环系统
- **60 FPS 实时渲染**：移动端高性能计算实现
- **论文数据一键导出**：支持 NetCDF & CSV 格式

## 🚀 核心功能
- **@ai-nowcast**：LSTM+Informer-Lite 轻量级预测模型
- **@gpu-eco**：移动端 GPU 加速，响应时间 <0.1ms
- **@storyline**：多风格叙事模式（科学/诗意/新闻）
- **@counterfactual**：反事实分析与情景模拟

## 📊 性能指标
- 4096 代理人 + 2048 云粒子：0.7 ms / step
- 包体 < 80 MB（含 LFS）
- 60 FPS 实测 OnePlus 11
- AI 预测：30分钟 <0.21ms，2小时 <0.15ms

## 🛠️ 架构优化
- EcoManager 统一管理四大核心模块
- 模块化设计，支持可选功能开关
- 自动 GPU 加速回退到 CPU

## 📚 论文支持
- 一键导出科研数据（NetCDF & CSV）
- SHAP 可解释性分析
